This folder contains:
* index.html - this is the entry point for viewing the technique in a browser.
* index.js - this is the entry point for running the technique in node.js.
* mnist_Module.js - this is the shared javascript module file that contains the generated code of your technique.

Both files contain "TODO:" comments listing what imported resources you need to provide, and optionally what variables you can modify.

For more information about WebGPU in Gigi, including limitations, running in node or the browser, please see UserDocumentation/WebGPU.docx or .pdf
